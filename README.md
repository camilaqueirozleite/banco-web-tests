Projeto de automação de testes end‑to‑end utilizando **Cypress**, validando funcionalidades essenciais de uma aplicação web bancária, como login, consulta de saldo e transferências.

Este projeto faz parte da minha formação prática como **QA em Testes Manuais, API e Automação**, aplicando boas práticas de qualidade, organização e escrita de testes.

---

## 🚀 Tecnologias utilizadas

- **Cypress** (E2E Testing)  
- **Node.js**  
- **JavaScript**  
- **Git & GitHub**  
- **Chrome DevTools**

## ▶️ Como executar o projeto

### 1. Clonar o repositório
git clone https://github.com/camilaqueirozleite/banco-web-tests.git

### 2. Instalar dependências
npm install

### 3. Abrir o Cypress
npx cypress open

### 4. Executar testes em modo headless
npx cypress run
